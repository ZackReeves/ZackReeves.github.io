Functions as a Service Simulation and Model

Project information

This project required building both a simulation and a model to analyse the performance of a Functions as a Service (FaaS) service. It uses the SimPy library to simulate a FaaS model and analyze the results based on a set of given input data. Investigation into how the cold start ratio and loss rate varied as the memory capacity of the system changed.

Simulation results were saved to csv files, and additional scripts were written to quickly analyse and plot the data produced.
